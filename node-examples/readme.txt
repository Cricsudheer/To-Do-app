How  to run  this  file  :
___________________________

1.Open  the file in any text  editor make sure   u  have installed  jscript  within  the  editor .
(if u are  using atom text  edito u can  install from packages named jscript)
2.Install packages  Express and ejs
	-npm install express
	-npm install ejs
3.Run file  using command   (make  sure u  have saved it )
	-node app
4.Use http://localhost:3000/ to open  the  home page
